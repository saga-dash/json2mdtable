# JSON2MDTableOfDefinition

## INSTALL

`npm install`

## BUILD

`npm run build`

## RUN

`node dist/app.js example/a.json`

## RESULT

```
## JSON

|Key|Description|Type|Formats|
|:-:|:-:|:-:|:-:|
|employees||array[Employees]||

## Employees

|Key|Description|Type|Formats|
|:-:|:-:|:-:|:-:|
|firstName||string||
|lastName||string||
```
